flask-georest

flask-georest è una web app che, per mezzo di dati spaziali e delle attuali tecnologie di posizionamento, permette l'accesso a dei documenti digitali a un dispostivo che si connette ad Internet attraverso un browser web

Se ti sei localizzato correttamente vedrai in rosso la regione effettiva in cui il voucher è disponibile.
